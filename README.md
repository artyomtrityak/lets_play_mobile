#React Native Flowtype Flux base setup

##Tech stack
- React Native
- Flowtype
- ESLint
- Watchman
- Flux

##How to setup
- `npm install`
- `brew install watchman` (if not installed yet with React Native)
- `npm start`
- `npm run watch`
- Open xcode project and launch Run

##How to stop
- `npm run stop`
